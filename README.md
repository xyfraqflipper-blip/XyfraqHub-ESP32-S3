# Xyfraq Hub - ESP32 Windows Remote Control

THIS PROJECT IS FOR EDUCATIONAL PURPOSES ONLY

Platform: ESP32

A powerful ESP32-based remote control system for Windows computers, featuring a modern web interface for system management and automation.

## Features

### System Controls
- Task Manager - Quick access to system monitoring
- PowerShell - Remote PowerShell access
- Run Dialog - Windows Run command interface
- System Management - Log off, shutdown, restart, and lock screen
- Command Prompt - Direct CMD access

### Windows Utilities
- Settings - Open Windows Settings
- Task View - Virtual desktop management
- File Explorer - Quick file system access
- Clipboard History - Windows clipboard manager
- Snipping Tool - Screen capture utility

### Window Management
- Window Controls - Maximize, minimize, and close windows
- Desktop View - Show desktop (Win+D)
- Quick Actions - Media controls, volume, and screenshots

### Web Interface
- Modern Dark Theme - Sleek, professional interface
- Responsive Design - Works on desktop and mobile
- Real-time Serial Monitor - Live debugging and monitoring
- Tab-based Navigation - Easy access to all features
- Visual Feedback - Animated button responses

## Hardware Requirements

- ESP32 Development Board
- Micro-USB Cable
- Computer with Windows OS
- WiFi Capability (for ESP32)
